# Decision Moments

Decision Moments is a digital publication focused on behavioral marketing, decision architecture, and the psychology of choice.  
It serves as a structured library of concepts, models, and applied frameworks for marketers, strategists, and product teams.

## Structure
The site is organized into four core sections:

- **Articles** — in‑depth explorations of behavioral concepts  
- **Topics** — curated collections of related ideas  
- **Guides** — practical, long‑form how‑to resources  
- **Glossary** — definitions of key behavioral terms  

All content is written in plain HTML and served via GitHub Pages.

## Purpose
To make behavioral marketing accessible, accurate, and applicable — without the fluff, hacks, or pop‑psych shortcuts.

## Status
Active development. New articles and sections are added regularly.
